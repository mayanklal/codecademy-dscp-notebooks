# Steps to check in code files.

### Step 1 - Intialize the git
`git init`

### Step 2 - Add Files
`git add .`

### Step 3 - Commit Files with Comment
`git commit -m "Comment"`

### Step 4 - Specify Branch
`git branch -M main`

### Step 5 - Push the files to Main Branch in Github
`git push -u origin main`
